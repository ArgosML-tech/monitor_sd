# Praman Prompts

Ready-to-use AI prompts for SAP UI5 test automation with Praman.
Copied to `praman-prompts/` in your project by `npx playwright-praman init`.

| Prompt              | File                          | Business Process    | Complexity |
| ------------------- | ----------------------------- | ------------------- | ---------- |
| Create Sales Orders | create-sales-orders.prompt.md | Order-to-Cash (O2C) | High       |

**To create your own prompt**: Copy `TEMPLATE.prompt.md` and fill in the placeholders.
**To modify a prompt**: Copy it to your own folder first — `init --force` will overwrite.
